# 세부 영어캠프 랜딩 페이지 기획안 (카카오채널 상담 전환)

## 1) 목표 및 핵심 메시지
- **목표**: 세부 영어캠프 관심자를 카카오채널 상담으로 전환하고, 적합한 유학원/캠프 매칭으로 이어지도록 한다.
- **핵심 메시지**:
  - “아이에게 맞는 세부 영어캠프를 3분 만에 찾고, 카카오채널로 바로 상담받으세요.”
  - “비용 효율 + 몰입 환경 + 안전 관리 = 세부 캠프가 유학의 첫걸음.”

## 2) 정보 구조 (IA)
1. **Hero**: 핵심 가치 제안 + 카카오채널 상담 CTA
2. **왜 세부인가**: 비용/몰입/안전/성과 근거
3. **캠프/유학원 비교**: 유형별 추천 + 비교표
4. **간단 테스트**: 5~7문항 → 유형 결과 + 추천 유학원/캠프 + CTA
5. **FAQ**: 학부모 우려 해소
6. **신뢰 요소**: 후기/수치/협력기관 로고
7. **하단 CTA**: 카카오채널 상담 유도

## 3) 페이지 카피 초안
### 3-1. Hero
- **헤드라인**: “우리 아이에게 딱 맞는 세부 영어캠프, 3분 만에 찾기”
- **서브카피**: “간단한 테스트로 캠프 유형을 추천받고, 카카오채널에서 바로 상담하세요.”
- **CTA 버튼**: “카카오채널 상담하기” / “캠프 유형 테스트 시작”

### 3-2. 왜 세부 영어캠프인가
- **섹션 타이틀**: “왜 세부인가요?”
- **포인트 1**: “합리적인 비용으로 영어 몰입 환경”
- **포인트 2**: “짧은 기간에도 체감 성과가 빠른 집중형 커리큘럼”
- **포인트 3**: “안전 관리와 생활 케어를 갖춘 캠프 옵션 다양”

### 3-3. 캠프/유학원 비교
- **섹션 타이틀**: “유형별 추천 유학원 한눈에 보기”
- **카드 예시**
  - **A형 (초급·회화 집중)**: “기초부터 말문 트기, 스피킹 집중 프로그램”
  - **B형 (학습 몰입형)**: “성적 향상 목표, 구조화된 커리큘럼”
  - **C형 (밸런스형)**: “학습+체험 균형, 처음 캠프 추천”

### 3-4. 테스트 CTA 섹션
- **타이틀**: “아이에게 맞는 캠프 유형을 3분 안에 찾기”
- **서브카피**: “테스트 결과에 따라 추천 유학원과 캠프를 알려드립니다.”
- **CTA**: “테스트 시작하기”

### 3-5. FAQ
- **Q1**: “초등학생도 가능한가요?”
- **Q2**: “안전 관리는 어떻게 되나요?”
- **Q3**: “비용은 어느 정도인가요?”

### 3-6. 하단 CTA
- **타이틀**: “지금 카카오채널에서 맞춤 상담 받기”
- **버튼**: “카카오채널 상담하기”
- **보조 카피**: “내 아이에게 맞는 캠프/유학원 추천까지 한 번에.”

## 4) 간단 테스트 로직 (퀴즈 설계)

### 4-1. 문항 설계 (예시 6문항)
1. 현재 영어 수준은? (초급/중급/상급)
2. 캠프 목표는? (회화 자신감/시험 대비/종합 성장)
3. 예산 범위는? (저가/중간/프리미엄)
4. 캠프 기간은? (2주/4주/8주 이상)
5. 학습 강도 선호는? (강도 높음/보통/체험 중심)
6. 숙소 선호는? (기숙사/가정형/무관)

### 4-2. 유형 분류 예시
- **A형: 스피킹 집중형**
  - 초급 + 회화 자신감 + 보통/체험 중심 선호
- **B형: 학습 몰입형**
  - 중급/상급 + 시험 대비 + 강도 높음 선호
- **C형: 밸런스형**
  - 목표 다양 + 보통 강도 + 체험 병행 선호

### 4-3. 매칭 결과 표시
- **결과 헤드라인**: “당신은 A형 (스피킹 집중형)입니다.”
- **추천 내용**:
  - 추천 유학원 2~3곳 카드
  - 추천 캠프 특징 요약
  - **CTA**: “카카오채널로 상담받기”
